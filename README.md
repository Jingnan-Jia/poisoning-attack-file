tf_get_nn_of_x.py
这个文件是完整的寻找ｘ（代码里写的x=x_test[0]）的最近邻的代码，同时包括了save_fig, save_figs两个完整的保存单张图片和多张图片的函数。代码描述：寻找最近邻的时候，用白化后的数据求概率向量，与ｘ的概率向量比较，对这些距离排序后，得到前N个数据的索引值，再从白化之前的原始图片中找出这些索引值对应的图片。
